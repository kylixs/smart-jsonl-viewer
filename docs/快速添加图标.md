# 快速添加图标指南

## 问题说明

Chrome 扩展可以不设置图标运行，会显示默认的 Chrome 扩展图标。如果想要自定义图标，可以按照以下方法操作。

---

## 方法一：在线生成图标（推荐，最快）

### 步骤 1：使用在线工具生成

访问以下任一网站：

1. **Favicon Generator**
   - 网址：https://realfavicongenerator.net/
   - 上传一张图片（建议 512x512 或更大）
   - 生成多种尺寸的图标
   - 下载压缩包

2. **Favicon.io**
   - 网址：https://favicon.io/
   - 可以用文字、表情符号或图片生成
   - 自动生成多种尺寸

### 步骤 2：准备图标文件

从下载的文件中，准备以下三个尺寸：
- `icon-16.png` (16x16 像素)
- `icon-48.png` (48x48 像素)
- `icon-128.png` (128x128 像素)

### 步骤 3：放置图标

```bash
# 将图标文件放到 public/icons/ 目录
cp icon-16.png public/icons/
cp icon-48.png public/icons/
cp icon-128.png public/icons/
```

### 步骤 4：更新 manifest.json

编辑 `public/manifest.json`，添加图标配置：

```json
{
  "manifest_version": 3,
  "name": "JSONL Viewer",
  "version": "1.0.0",
  "description": "智能 JSONL 查看器，支持字符串解码和高级过滤",
  "icons": {
    "16": "icons/icon-16.png",
    "48": "icons/icon-48.png",
    "128": "icons/icon-128.png"
  },
  "permissions": ["storage"],
  // ... 其余配置保持不变
  "action": {
    "default_popup": "index.html",
    "default_icon": {
      "16": "icons/icon-16.png",
      "48": "icons/icon-48.png",
      "128": "icons/icon-128.png"
    }
  }
}
```

### 步骤 5：重新构建

```bash
npm run build
```

### 步骤 6：在 Chrome 中刷新扩展

1. 访问 `chrome://extensions/`
2. 找到 JSONL Viewer
3. 点击刷新图标 🔄

## 方法二：使用表情符号快速生成

### 使用 Favicon.io 的 Emoji 功能

1. 访问：https://favicon.io/emoji-favicons/
2. 选择一个相关的表情符号，比如：
   - 📄 文档
   - 📋 剪贴板
   - 🗂️ 文件夹
   - 💾 保存
   - 📊 图表
3. 下载生成的图标包
4. 按照方法一的步骤 2-6 操作

## 方法三：设计建议

如果你想设计专业的图标，建议使用：

### 设计元素
- **主题**：JSON/JSONL 相关
- **颜色**：与扩展界面配色一致（紫色渐变）
- **符号**：
  - `{ }` 大括号
  - `[ ]` 方括号
  - 文档/文件图标
  - 代码编辑器图标

### 推荐工具
- **Figma** - 在线设计工具
- **Canva** - 简单易用
- **Photoshop/Illustrator** - 专业设计

### 设计规范
- **尺寸**：建议从 512x512 开始设计
- **格式**：PNG 24-bit（支持透明）
- **背景**：建议使用透明或纯色
- **简洁性**：在小尺寸下依然清晰

## 临时方案：不设置图标

如果暂时不想设置图标，当前的配置已经可以正常工作：
- ✅ 扩展功能完全正常
- ✅ 会显示 Chrome 默认的拼图图标
- ✅ 不影响使用

等以后有合适的图标再添加即可。

## 示例：快速生成紫色 JSON 图标

如果你想要一个简单的图标，可以使用以下步骤：

1. 访问 https://favicon.io/favicon-generator/
2. 设置如下：
   - **Text**: `{}`
   - **Background**: `Rounded`
   - **Font Family**: `Courier New`
   - **Font Size**: `70`
   - **Font Color**: `#FFFFFF`
   - **Background Color**: `#667EEA` (扩展主题色)
3. 点击 Download
4. 将下载的文件重命名并放到 `public/icons/`
5. 更新 manifest.json
6. 重新构建

## 常见问题

### Q: 图标没有显示？
A: 确保：
1. 文件名完全匹配（`icon-16.png` 等）
2. 文件在正确的位置（`public/icons/`）
3. 重新构建了项目（`npm run build`）
4. 在 Chrome 中刷新了扩展

### Q: 图标显示模糊？
A: 确保：
1. 图标是 PNG 格式
2. 尺寸完全匹配要求（16x16, 48x48, 128x128）
3. 使用高质量的原始图片

### Q: 可以使用 SVG 吗？
A: Chrome 扩展图标必须使用 PNG 格式，不支持 SVG。
