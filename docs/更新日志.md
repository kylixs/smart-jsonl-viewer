# 更新日志

## [1.1.0] - 2026-01-04

### ✨ 新增功能

#### 1. 新标签页打开
- **优化前**：点击扩展图标打开小弹窗
- **优化后**：点击扩展图标在新标签页打开，占满整个浏览器窗口
- **优势**：更大的查看空间，更好的使用体验

#### 2. 展开深度控制
- **新增深度选择器**：位于工具栏，可选择展开深度
- **支持深度**：
  - 全部折叠（0 层）
  - 1-5 层展开
  - 全部展开
- **智能展开**：自动展开到指定层级，超过深度的节点保持折叠
- **用途**：快速浏览大型 JSON 文件的结构，避免展开过多导致页面卡顿

#### 3. 复制转码后内容
- **字符串复制**：解码字符串旁边添加 📋 复制按钮
  - 点击复制解码后的文本内容
  - JSON 字符串会格式化后复制
  - 显示 ✓ 确认复制成功
- **整行复制**：每行 JSON 右侧悬停显示 📋 复制按钮
  - 复制整行的解码内容（包括所有嵌套字符串）
  - 自动格式化为可读的 JSON
  - 2 秒后自动隐藏成功提示

### 🎨 UI 改进

- 深度选择器采用半透明背景，与工具栏主题色融合
- 复制按钮采用蓝色配色，与主题一致
- 行级复制按钮悬停时才显示，避免界面过于拥挤
- 复制成功后显示 ✓ 图标，提供即时反馈

### 🔧 技术优化

- 优化 JsonTree 组件，支持深度控制
- 改进 background.ts，监听图标点击事件
- 移除 popup 模式，改用标签页模式
- 添加 `decodeObject` 导出，供复制功能使用

## 版本说明

### 版本号规则
遵循语义化版本控制（Semantic Versioning）：
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 版本发布流程
1. 更新版本号（package.json + manifest.json）
2. 更新此文档
3. 运行 `npm run build`
4. 测试新功能
5. 提交代码
6. 打 tag 并发布
